# Security Policy

## Local-first design

AI Dev Brain Kit runs entirely on your machine. No data leaves your system unless you explicitly enable LLM integration features.

## Reporting vulnerabilities

If you discover a security issue, please use [GitHub Security Advisories](https://github.com/MohamedHussien-zseeker/ai-dev-brain-kit/security/advisories/new) to report it privately.

For non-sensitive bugs, you can also open a [GitHub issue](https://github.com/MohamedHussien-zseeker/ai-dev-brain-kit/issues).

## Scope

This security policy applies to the AI Dev Brain Kit binary and CLI. It does not cover third-party services you connect to (LLM APIs, Obsidian, etc.).
