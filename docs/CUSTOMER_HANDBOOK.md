# Customer Handbook

## What Is Brain CLI?

Brain CLI is a local-first memory system for solo developers. It gives you a structured vault for notes, decisions, daily logs, and knowledge — plus tools to consolidate and review your raw captures into durable references.

## What's Included (All Packages)

- Prebuilt binaries for Linux and Windows
- Install scripts with SHA-256 verification
- Vault templates (daily log, project, decision, prompt, session)
- CLI commands: `note`, `today`, `context`, `consolidate`, `review`, `doctor`
- Claude Code integration via `brain hook install`
- Documentation: Quickstart, Privacy, Security, Uninstall

## What You Own

- All your data stays on your machine
- The software is proprietary — see LICENSE in the repository
- No subscriptions, no accounts, no telemetry
- Perpetual license (Starter+): use the version delivered forever

## AI Options

Brain works with any OpenAI-compatible provider:
- **Cloud AI** (default) — bring your own API key (OpenAI, Anthropic via API, etc.)
- **Local AI** (Premium) — Ollama, LM Studio, fully offline
- **No AI** — heuristic-only consolidation, no data leaves your machine

Brain never bundles or resells AI access. You control your key and your data.

## What Brain Does Not Do

- Does not provide perfect memory — it stores what you capture
- Does not offer unlimited support — each package has defined bounds
- Does not auto-install — your engineer sets everything up
- Does not share your data — no telemetry, no analytics, no cloud sync
