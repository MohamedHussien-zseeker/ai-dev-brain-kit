# Customer Handbook

## What Is Brain CLI?

Brain CLI is a local memory system for solo developers. It gives you a structured vault for notes, decisions, daily logs, and knowledge — plus tools to consolidate and review your raw captures into durable references.

## What You Get

### The Setup Service ($299)

Your purchase includes one 60-minute remote session where I:

1. **Audit your current workflow** — note-taking, context building, AI tool usage
2. **Install and configure** Brain CLI on your machine(s)
3. **Set up your vault** — directory structure, templates, config
4. **Connect your LLM provider** (optional) — OpenAI, Ollama, or any OpenAI-compatible endpoint
5. **Install the Claude Code stop hook** — auto-capture session summaries
6. **Walk through the daily workflow** — capture, consolidate, review
7. **Deliver a written setup summary** — what was installed, where, and how to use it

### What's Included

- Prebuilt binaries for Linux and Windows
- Install scripts with SHA-256 verification
- Vault templates (daily log, project, decision, prompt, session)
- CLI commands: `note`, `today`, `context`, `consolidate`, `review`, `doctor`
- Claude Code integration via `brain hook install`
- Documentation: Quickstart, Privacy, Security, Uninstall

### What You Own

- All your data stays on your machine
- The software is open source (MIT)
- No subscriptions, no accounts, no telemetry
